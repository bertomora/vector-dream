### Technical Decisions Made Today
1. **Hash-based seeds (#seed=)**: CDNs strip query params but preserve URL hashes
2. **gateway.irys.xyz**: More reliable than arweave.net/ar-io.net
3. **V1_LEGACY_TOKENS array**: [3,4,5,6,7] in metadata API
4. **Iframe detection**: `window.self !== window.top` adds `.embedded` class to hide UI when embedded in OpenSea
